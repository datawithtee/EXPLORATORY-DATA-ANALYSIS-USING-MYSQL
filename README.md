# EXPLORATORY-DATA-ANALYSIS-USING-MYSQL
Exploratory Data Analysis (EDA) of a sales dataset using MySQL. The project covers data cleaning, SQL queries, KPI calculations, and business insights.
# Overview

This project presents an Exploratory Data Analysis (EDA) of a sales dataset using MySQL. The analysis focuses on cleaning the data, writing SQL queries, calculating key business metrics, and extracting insights to support data-driven decision-making.

# Objectives
Import and clean the sales dataset in MySQL.
Explore sales and customer data using SQL queries.
Calculate key performance indicators.
Identify top-performing products and customers.
Analyze sales trends and order statuses.

# Tools Used
MySQL
CSV Dataset

# SQL Analysis Performed
Total number of orders.
Total sales generated.
Average order value.
Quantity sold analysis.
Product performance analysis.
Sales by payment method.
Order status distribution.
Monthly sales trend analysis.
Top customer analysis.

# Key Performance Indicators (KPIs)
Total Sales
Total Orders
Average Order Value
Total Quantity Sold
Top Products
Top Customers
Order Status Metrics

# Key Insights
Product categories contributed differently to overall revenue.
Customer spending patterns varied across the dataset.
Order cancellations and returns affected total performance.
Sales trends provided valuable information for business planning.
